# coc-github-comments

coc-extension to display GitHub line comments in NeoVim.

## Install

`:CocInstall coc-github-comments`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-github-comments-keymap)`

## Lists

`:CocList demo_list`
